Abstract

Predicting stock prices is a challenging task due to the complexity and volatility of financial markets. In this project, we propose a hybrid approach that combines technical indicators derived from historical candlestick patterns with sentiment analysis of news data to predict future stock prices.

Our approach uses polynomial linear regression on a transformed price series (calculated from high/low averages and rolling means) and integrates this with sentiment classification outputs to account for real-world influences. By merging two distinct models — one for numerical prediction and one for sentiment understanding — we aim to improve prediction accuracy and market responsiveness.

Dataset

The dataset used for this project is a CSV file named All_new.csv which includes:

  • Open, High, Low, Adj Close values for each day
  • A new calculated Price = (High + Low) / 2
  • Rolling Average column: a 10-day rolling mean of prices
This structured dataset serves as the basis for our first model's time-series forecasting, providing a clearer picture of market trends.

Models Used

We experimented with several machine learning algorithms for both components of the system:

Model 1: Price Prediction

  • KNN Regressor
  • Decision Tree Regressor
  • Polynomial Linear Regression (Best Performer)

Model 2: Sentiment Analysis

  • SVM Classifier
  • KNN Classifier
  • Decision Tree Classifier (Best Performer)

We combined these models to create a hybrid predictor where sentiment classification adjusts or influences the regression output.

Results

The final merged model outperformed standalone approaches in both stability and accuracy. Key results:

  • Polynomial regression accurately captured price trends over time
  • Sentiment analysis flagged negative news that correlated with dips
  • Hybrid predictions showed improved Mean Squared Error (MSE) scores
